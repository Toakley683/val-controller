# README

## About

This is a tool I've made for Valorant tracking, I didn't like any of the available trackers, so I made my own which was lightweight and functional

## Features

- Skin Loadouts
- Better Skin Randomization
- Random Agent Selector
- Exit Pregame early
- See skins of other players
- See current/peak rank of players

## Installation

Goto the releases tab and download the installer in there
